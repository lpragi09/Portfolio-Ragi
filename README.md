# Portfólio (Estilo Ragi) — Next.js + Tailwind

Este repositório contém:

- **`/legacy`**: versão antiga (HTML/CSS/JS) preservada
- **`/src`**: versão atual em **Next.js 14+ (App Router) + TypeScript + Tailwind**

## Rodar localmente

```bash
npm install
npm run dev
```

## Variáveis de ambiente (Supabase)

Crie um `.env.local` a partir de `env.example` (não commitar chaves):

```bash
cp env.example .env.local
```

## Checklist de stack (obrigatório)

- Next.js 14+ (App Router) + TypeScript
- Tailwind CSS
- Lucide-react
- Estrutura preparada para Supabase
- Vercel Analytics + Speed Insights em `src/app/layout.tsx`
